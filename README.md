# ☕ Dejai Brew – Digital Coffee Kiosk

Welcome to **Dejai Brew**, a modern interactive coffee shop kiosk built using HTML, CSS, and JavaScript.  
This project simulates a real café ordering system with drinks, snacks, animations, and a dynamic cart system.

---

## 🚀 Features

### ☕ Menu System
- Coffee drinks (Espresso, Latte, Cappuccino, etc.)
- Cold brew options
- Signature drinks
- 🍪 Snacks tab (croissants, muffins, brownies, cookies)

### 🧠 Interactive UI
- Click-to-add ordering system
- Dynamic cart with live total updates
- Category filtering tabs (Coffee / Cold / Signature / Snacks)

### ✨ Animations
- Smooth hover effects on menu cards
- Liquid “fill” animation when adding items
- 3D-style card interactions (optional extension)
- Steam/motion-style UI effects (upgrade-ready)

---

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox + Grid + Animations)
- Vanilla JavaScript (no frameworks)
